AI tools connected to your [doc MCP server](/help/documentation-experience/ask-ai-capabilities/#mcp-server) can now answer questions like "What changed recently on this API?", straight from your documentation changelog. Two new tools are available: 
- **list_changes**: browse the changelog of a documentation.
- **get_change**: retrieve the full content of a specific change.

The doc MCP server runs on top of the [Portal API](https://developers.bump.sh/doc/portal), meaning these two tools are also exposed as endpoints, if you need to access the changelog programmatically.

As always, don’t hesitate to [reach out](mailto:hello@bump.sh) if you have any feedback.
