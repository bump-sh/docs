We released our new search experience a few months ago. We made many adjustments and improvements, in both results relevancy and user experience, thanks to your feedback.

**Search scopes, our last release, makes it easier for your users to discover your API catalog:** with a one-click switch from a doc-only search to a global hub-search, your users no longer get stuck with an empty search without follow-up actions.

![Search scopes changed to "Search in all APIs" after an empty results list](/docs/images/changelog/search-scopes.gif)

You can change the default search scope for an API (current API or global) in your hub settings. By default, "Search in all APIs" is triggered when clicking the checkbox in the search modal.

![Search scopes setting in hub settings](/docs/images/changelog/search-scopes-setting.png)
