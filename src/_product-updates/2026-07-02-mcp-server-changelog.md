---
title: Changelog access for your docs MCP server
tags: [New]
---

AI tools connected to your [doc MCP server](/help/documentation-experience/ask-ai-capabilities/#mcp-server) can now answer questions like "what changed recently on this API?", straight from your documentation changelog.

This is powered by two new endpoints on the [Portal API](/help/documentation-experience/ask-ai-capabilities/), which the MCP server runs on top of:
- **[List changes](https://developers.bump.sh/doc/portal/operation/operation-get-changes-list)**: browse the changelog of a documentation.
- **[Get a change](https://developers.bump.sh/doc/portal/operation/operation-get-changes-fetch)**: retrieve the full content of a specific change.
